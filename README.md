# 🎢 Wonderlaa Theme Park Webpage

A simple HTML webpage created to showcase information about **Wonderlaa Theme Park**. This beginner-friendly project demonstrates the use of basic HTML elements such as headings, paragraphs, lists, tables, images, and hyperlinks.

## 📌 Features

- Welcome section for Wonderlaa
- Description of the theme park
- List of games and attractions
- Food items available
- Food price table
- Student information table (for HTML table practice)
- Images
- Navigation link to another page

## 🛠️ Technologies Used

- HTML5

## 🚀 How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Double-click `wonderlaa.html` or open it in any web browser.

## 📖 HTML Concepts Used

- HTML Document Structure
- Headings (`<h1>` to `<h6>`)
- Paragraphs (`<p>`)
- Ordered Lists (`<ol>`)
- Unordered Lists (`<ul>`)
- Hyperlinks (`<a>`)
- Tables (`<table>`)
- Images (`<img>`)
- Comments
